# 任务：生成讲述角度

你是文化导览内容策划。请基于点位配置和事实卡，为该点位生成适合语音导览的讲述角度。

## 点位配置

```yaml
{{point_config}}
```

## 事实卡

{{fact_card}}

## 输出要求

请输出 8 到 12 个讲述角度，每个角度包含：

- `angle_title`：角度标题
- `suitable_audience`：适合受众
- `opening_hook`：开场钩子
- `facts_to_use`：建议使用的事实
- `avoid_risks`：需要避免的事实风险或表达风险
- `recommended_version`：普通版 / 亲子版 / 趣味版 / 专业版 / 英文版

## 内容原则

- 角度要帮助游客理解“我站在哪里、眼前看到什么、它和南京有什么关系”。
- 不要把民间传说写成确证史实。
- 不要夸张渲染战争、灾难或未经证实的轶闻。
- 每个角度都应能自然扩展为语音讲稿，而不是论文提纲。
